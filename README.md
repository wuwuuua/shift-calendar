# 个人排班日历

一个简洁实用的个人排班管理工具，支持可视化日历操作和 ICS 日历订阅功能。

## 功能特性

- 📅 **可视化日历界面** - 直观的月历视图，方便查看和设置排班
- ⌨️ **键盘快捷键** - 快速设置排班状态，提升操作效率
- 📤 **ICS 导出** - 导出标准日历文件，导入到系统日历或其他日历应用
- 🔔 **自动提醒** - 为每个班次自动添加 35 分钟前的闹钟提醒
- 💾 **本地存储** - 数据保存在浏览器本地存储中，无需服务器
- 🎨 **现代化界面** - 简洁美观的用户界面设计
- 📊 **统计功能** - 自动统计各班次数量

## 班次类型

| 班次 | 名称 | 时间 |
|------|------|------|
| D | 早班 | 08:00 - 16:00 |
| S | 中班 | 16:00 - 24:00 |
| N | 夜班 | 00:00 - 08:00 |
| W | 休息 | - |

## 快捷键

选中日期后，使用以下快捷键快速设置排班：

- `D` - 设置为早班
- `S` - 设置为中班
- `N` - 设置为夜班
- `W` - 设置为休息
- `Delete` / `Backspace` - 清除排班

## 使用方法

### 基本使用

1. 用浏览器打开 `index.html`
2. 点击日期选中，然后使用快捷键设置排班
3. 或者点击已有排班进行切换（D → S → N → W → 无）
4. 点击"导出 ICS"按钮导出日历文件

### 导入到日历

1. 点击"导出 ICS"按钮下载 `.ics` 文件
2. 在日历应用中选择"导入"或"订阅"
3. 选择下载的 `.ics` 文件

**支持的平台：**
- macOS 日历
- Google Calendar
- Outlook
- 手机系统日历（iOS/Android）

## 技术栈

- 纯 HTML / CSS / JavaScript
- 无需后端服务器
- 使用 LocalStorage 进行数据持久化

## 本地运行

```bash
# 克隆仓库
git clone <repository-url>

# 进入目录
cd 排班ics日历订阅

# 用浏览器打开 index.html
open index.html  # macOS
# 或双击 index.html 文件
```

## 数据管理

- **清空数据**：点击右上角"清空数据"按钮可清除所有排班记录
- **数据备份**：建议定期导出 ICS 文件作为备份
